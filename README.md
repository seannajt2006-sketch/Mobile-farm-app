# FarmConnect

Mobile app connecting farmers and buyers with buyer/seller/admin flows.

## Structure
- frontend/ — React Native (Expo, TypeScript)
- backend/ — FastAPI, PostgreSQL, Cloudinary image URLs
- docs/ — dev log, design notes, API spec

## Quick start
- Frontend: see frontend/README.md
- Backend: see backend/README.md

## Roles & features
- Buyers: browse products, send order requests
- Sellers: add/manage products with images
- Admins: review and approve/block listings/users

## Status
Actively developed; follow docs and prototypes under Prototype/ for UI references.
